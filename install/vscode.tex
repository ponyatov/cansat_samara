\bigskip\noindent
На некоторых компьютерах с \win\ возникает проблема с запуском\ --- черный экран. В этом случае \vsc\ придется
запускать из \file{.bat}-файла или через \keys{\winstart+R}:

\begin{verbatim}
C:/cansat_samara> code --disable-gpu \cansat_samara
\end{verbatim}
